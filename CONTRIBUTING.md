# Panduan Kontribusi (Contributing Guidelines)

Kami senang Anda tertarik untuk berkontribusi pada proyek PjBL Akademik ini! Proyek ini dikembangkan sebagai bagian dari tugas mata kuliah **Cybersecurity Dan Threat Intelligence (2025T2-SIF25606T) UTDI**.

## Bagaimana Anda Dapat Berkontribusi?

### 1. Melaporkan Masalah (Reporting Issues)
Jika Anda menemukan kesalahan ketik dalam draf laporan, ketidaksesuaian teori, atau masalah teknis pada slide presentasi web (misalnya tata letak slide rusak di perangkat tertentu), silakan buat *Issue* baru di repositori GitHub ini.

### 2. Mengajukan Perubahan (Submitting Pull Requests)
Untuk mengajukan perbaikan atau penambahan konten:
1. Lakukan **Fork** pada repositori ini.
2. Buat cabang baru (*branch*) untuk fitur atau perbaikan Anda:
   ```bash
   git checkout -b perbaikan/tata-bahasa-bab-4
   ```
3. Lakukan commit perubahan Anda dengan pesan yang deskriptif dan sopan:
   ```bash
   git commit -m "Memperbaiki salah ketik kata triase di Bab IV"
   ```
4. Push cabang Anda ke repositori fork Anda:
   ```bash
   git push origin perbaikan/tata-bahasa-bab-4
   ```
5. Buat **Pull Request** (PR) baru ke cabang utama (*main*) repositori kami.

### 3. Pedoman Konten Akademik
Karena ini adalah proyek akademik UTDI untuk mata kuliah Cybersecurity & Threat Intelligence:
* Pastikan setiap penambahan data atau studi kasus baru menyertakan sumber referensi ilmiah yang valid.
* Hindari plagiarisme. Tuliskan rujukan secara jelas menggunakan gaya penulisan referensi UTDI.

Terima kasih atas kontribusi Anda!
