# Analisis Dampak AI/LLM Terhadap Kesehatan Mental Developer Open Source

Repositori ini berisi proyek pembelajaran berbasis proyek (**Project-Based Learning - PjBL**) untuk menganalisis pergeseran beban kerja dan implikasi kesehatan mental (*burnout*, stres, kecemasan) yang dihadapi oleh kontributor open source akibat maraknya laporan celah keamanan otomatis berbasis kecerdasan buatan (*AI/LLM-generated security reports*).

## 💻 Cara Menjalankan Slide Presentasi
Slide presentasi dikembangkan sebagai aplikasi satu halaman (*single-page web application*) yang sepenuhnya mandiri (*self-contained*).
1. Unduh repositori ini ke komputer Anda.
2. Masuk ke folder proyek.
3. Klik ganda (double-click) pada berkas `index.html`.
4. Presentasi akan otomatis terbuka di browser Anda.

### Kontrol Navigasi Slide:
* ➡️ **Panah Kanan** / **Spasi**: Pindah ke slide berikutnya.
* ⬅️ **Panah Kiri**: Pindah ke slide sebelumnya.
* 📱 **Layar Sentuh**: Geser layar (*swipe* kiri/kanan) jika dibuka dari smartphone.
* 🖥️ **Tombol F**: Masuk/keluar dari mode layar penuh (*fullscreen*).

## 📝 Ringkasan Hasil Kajian (PjBL)
AI/LLM telah mendemokratisasi penemuan bug keamanan secara instan, namun ini memicu ketimpangan kerja (*asymmetry of labor*):
* **Sisi Ofensif (Pencari Bug)**: Cukup mengeluarkan biaya murah (< $20) dan menulis skrip shell pendek untuk memindai ribuan file secara otomatis.
* **Sisi Defensif (Maintainer)**: Harus memverifikasi secara manual setiap laporan crash, memilah laporan sampah/halusinasi AI, menulis *patch*, serta menghadapi argumen buatan LLM dari pelapor nakal. Hal ini memicu kelelahan kognitif kronis, rasa cemas, dan risiko *burnout* tinggi bagi maintainer yang bekerja sukarela.
